0. 사전 정의: 입력 데이터 구조

이 문서의 모든 시스템은 설문 결과로부터 도출된 8축 성향 카운트(axes)를 입력으로 받습니다.

0-1. axes 객체

// 각 사용자별 1개씩 생성됨
axes = {
  개방성: 0,    // 새로운 경험, 호기심, 창의성
  성실성: 0,    // 계획, 책임감, 목표 달성 능력
  외향성: 0,    // 사교성, 활동 에너지, 활력
  우호성: 0,    // 타인 배려, 공감 능력, 협조성
  정서안정: 0,  // 스트레스 관리, 평정심 유지
  애착안정: 0,  // 관계에 대한 신뢰와 편안함
  애착불안: 0,  // 사랑과 관심을 확인받고 싶음
  애착회피: 0   // 독립적인 시간과 공간 중시
}

각 설문 문항의 선택지에 #개방성, #성실성 등의 태그가 매핑되어 있으며, 사용자가 선택한 옵션의 태그를 축(axis)으로 변환하여 1씩 카운트합니다.

0-2. SLOT_COUNT — 이게 뭐고 왜 필요한가?

핵심 개념: 24개 설문 문항 전체에서 각 축(axis)에 매핑되는 선택지의 총 개수가 SLOT_COUNT입니다.

예를 들어 성실성의 SLOT_COUNT가 18이라는 건, 24문항 전체를 통틀어 #성실성으로 매핑되는 선택지가 총 18개 존재한다는 뜻입니다. 한 사람이 모든 문항에서 성실성 태그가 붙은 옵션만 골랐다면 최대 18점이 나옵니다.

왜 필요한가? — 축 간 공정 비교를 위해서입니다.

애착불안은 슬롯이 6개밖에 안 되고 성실성은 18개나 되므로, 단순 카운트(raw count)로 비교하면 성실성이 항상 높게 나옵니다. 그래서 지배적 성향(dominantTrait)을 결정할 때 count / SLOT_COUNT로 정규화해서 비율로 비교합니다.

// 정규화 예시
norm_성실성  = axes['성실성'] / 18  →  예: 6/18 = 0.33
norm_애착불안 = axes['애착불안'] / 6   →  예: 4/6  = 0.67

// → raw count는 성실성(6)이 높지만,
//   정규화하면 애착불안(0.67)이 지배적 성향이 됨
축SLOT_COUNT설명
개방성12호기심의 색
성실성18신뢰의 색
외향성10활력의 색
우호성10배려의 색
정서안정9평온의 색
애착안정13깊은 신뢰의 색
애착불안6감수성의 색
애착회피6자존의 색

0-3. 지배적 성향(Dominant Trait) 결정

function getDominantTrait(axes) {
    let maxNorm = -1, dom = '우호성';
    for (const [trait, score] of Object.entries(axes)) {
        const norm = score / SLOT_COUNT[trait];  // ← 정규화
        if (norm > maxNorm) { maxNorm = norm; dom = trait; }
    }
    return dom;
}

이 지배적 성향이 파동 색상, 매칭 타입, 결 해석 프로필 전부에 핵심 입력으로 사용됩니다.

1. 파동 표현 (Wave Visualization)

Canvas 위에 사인(sin) 파형을 실시간 애니메이션으로 렌더링합니다. 남녀 각각 별도 파형으로 표시되며, 3축(진폭/주파수/위상)이 성격을 시각적으로 표현합니다.

1-1. 3축 계산 공식

① 진폭 (Amplitude) — 관계 에너지 크기

파동의 높이. 높을수록 관계에 적극적이고 에너지가 강합니다.

raw = 외향성 × 1.2 + 개방성 × 0.6 + 애착안정 × 0.4 - 애착회피 × 0.8
level = clamp(round(raw + 0), 1, 10)           // offset = 0
pixelAmp = 8 + (level - 1) / 9 × (maxAmplitude - 8)
// maxAmplitude = (canvasHeight / 2) - 60

② 주파수 (Frequency) — 감정 리듬

파동이 얼마나 빨리 반복되는지. 높을수록 감정 변화가 빠르고 변동적입니다.

raw = 애착불안 × 1.0 + 개방성 × 0.4 - 정서안정 × 0.8 - 성실성 × 0.3
level = clamp(round(raw + 6), 1, 10)           // offset = 6
pixelFreq = 0.008 + (level - 1) / 9 × 0.027

③ 위상 (Phase) — 리드/팔로우 역할

두 파동 간의 시간차. 차이가 클수록 한 사람이 이끌고 한 사람이 따르는 구조입니다.

raw = 외향성 × 0.8 + 애착안정 × 0.5 - 우호성 × 0.9 - 애착회피 × 0.7
level = clamp(round(raw + 5.5), 1, 10)         // offset = 5.5
phaseDiff = (mPhaseLevel - fPhaseLevel) × 0.3  // 남녀 위상차

1-3. Canvas 렌더링 의사코드

function animate() {
  ctx.clearRect(0, 0, width, height);
  
  // ① 남성 파형
  for (x = 0 to width) {
    y = centerY + sin(x × freqM + offset) × ampM
  }
  stroke: LinearGradient(보조색 → 주색 → 보조색), lineWidth=4, shadowBlur=15
  
  // ② 여성 파형 (위상차 적용)
  for (x = 0 to width) {
    y = centerY + sin(x × freqF + offset + phaseDiff) × ampF
  }
  stroke: LinearGradient(보조색 → 주색 → 보조색), lineWidth=4, shadowBlur=15
  
  offset += 0.03;  // 애니메이션 속도
  requestAnimationFrame(animate);
}

3. 결 해석 (Gyeol Interpretation)

각 개인의 성격 프로필 + 커플 종합 해석을 텍스트로 생성합니다.

3-1. 개인 프로필 생성 흐름

Step 1) 모든 8축을 정규화(count / SLOT_COUNT)하여 내림차순 정렬 → top1, top2 결정
Step 2) 3축 레벨(진폭/주파수/위상) 계산 → 사분면(Quadrant) 결정
        - ampLevel >= 6 → 에너지높음(H), < 6 → 낮음(L)
        - freqLevel <= 5 → 안정적(H),   > 5 → 변동적(L)
        - 조합: HH / HL / LH / LL
Step 3) profiles[top1+top2][quad] → 프로필 한 줄 설명
Step 4) 3축 레벨별 10단계 설명 + 색상 해석 조합
사분면조건의미이모지
HH에너지높음 + 안정적적극적이면서도 안정적인 사람⚡
HL에너지높음 + 변동적에너지가 높고 감정도 풍부한 사람🔥
LH에너지낮음 + 안정적차분하고 한결같은 사람🌿
LL에너지낮음 + 변동적조용하지만 내면이 깊은 사람🌙

3-2. 3축 10단계 설명 전체 (30개)

⛰️ 진폭 (관계 에너지) — 1~10

Lv설명
1관계에서 매우 조용한 편이에요. 가까운 사이에서도 표현보다 마음속에 담아두는 타입이에요.
2표현이 적지만 진심이 담긴 사람이에요. 소수의 사람에게만 마음을 열어요.
3차분하게 관계를 유지해요. 조용하지만 함께 있으면 편안한 안정감이 느껴져요.
4부드럽고 온화한 에너지예요. 상대방 페이스에 맞춰주며 편안한 분위기를 만들어요.
5상황에 따라 적극적이기도, 조용하기도 해요. 관계에서 유연하게 에너지를 조절해요.
6관계에 따뜻한 에너지를 불어넣어요. 함께 있으면 기분이 좋아지는 사람이에요.
7관계에 적극적으로 참여해요. 먼저 연락하고, 만남을 즐기는 행동력이 있어요.
8넘치는 에너지로 주변을 밝혀요. 함께 있으면 활력이 전해지는 사람이에요.
9관계에 온 마음을 쏟는 사람이에요. 사랑하는 사람에게 아낌없이 에너지를 줘요.
10최고 수준의 관계 에너지예요. 강렬한 애정 표현과 적극적인 참여가 특징이에요.

🎵 주파수 (감정 리듬) — 1~10

Lv설명
1감정이 거의 흔들리지 않아요. 언제 만나도 한결같은 모습이에요.
2감정 기복이 적고 꾸준해요. 약속과 계획을 잘 지키는 믿음직한 사람이에요.
3대체로 안정적이에요. 가끔 감정이 드러나지만 빠르게 평정심을 찾아요.
4일관된 태도를 유지하면서도 필요할 때 유연하게 반응해요.
5안정과 변화 사이에서 균형을 잡아요. 상황에 따라 다른 모습을 보여요.
6변화에 잘 적응하고 다양한 반응을 보여요. 같은 상황에서도 다른 매력을 발견할 수 있어요.
7감정 표현이 다채로워요. 기쁨도 슬픔도 깊이 느끼고 솔직하게 드러내요.

4-1. Big5 성향별 설명 (순위 기반)

각 사용자의 Big5 점수를 높은 순으로 정렬하여 Top 3를 표시합니다. 1순위(dominant)와 2~3순위(supportive)에 따라 다른 설명이 출력됩니다.

성향유형레이블설명
개방성1순위🔭 혁신적인 탐험가귀하를 대표하는 키워드는 '통찰'과 '혁신'. 남들이 보지 못하는 가능성을 발견하고, 새로운 경험을 통해 끊임없이 성장하려는 욕구가 매우 강합니다.
2~3순위✨ 유연한 수용가열린 사고방식을 가지고 있어 변화에 유연하게 대처합니다. 당신의 다른 강점들에 창의적인 색채를 더해주는 역할을 합니다.
성실성1순위📅 목표 지향적 설계자귀하의 가장 큰 무기는 '철저함'. 삶의 불확실성을 계획으로 통제하며, 높은 책임감으로 목표를 반드시 달성해내는 성취가입니다.
2~3순위🏃 신뢰할 수 있는 실행가성실함이 귀하의 기반을 단단하게 받쳐줍니다. 맡은 바 책임을 다하는 꾸준함으로 파트너에게 깊은 신뢰와 안정감을 줍니다.
외향성1순위🗣️ 관계 지향적 리더외부 세계와의 상호작용에서 에너지를 얻는 타입. 적극적인 태도로 관계를 주도하며, 타인에게 긍정적인 활력을 불어넣는 중심인물입니다.
2~3순위☀️ 활력 있는 사교가적당한 사교성을 갖추어 원만한 대인 관계를 형성합니다. 상황에 따라 에너지를 발산하며 분위기를 밝게 만드는 윤활유 역할을 합니다.
우호성1순위🤝 이타적인 조력자타인에 대한 깊은 이해와 배려심이 핵심. 자신의 이익보다 공동체의 조화와 파트너의 행복을 우선시하는 따뜻한 성품을 지녔습니다.
2~3순위🕊️ 온화한 중재자공감 능력이 뛰어나 타인의 감정을 잘 읽습니다. 갈등 상황에서도 부드러운 태도로 관계의 균형을 맞추는 능력이 탁월합니다.
정서안정1순위⚓️ 흔들림 없는 중심스트레스 내성이 매우 높고 감정 기복이 적습니다. 어떤 위기 상황에서도 이성적이고 침착하게 대처하여 주변에 강력한 심리적 안정을 제공합니다.
2~3순위🌊 평온한 유지자쉽게 예민해지지 않는 무던한 성격. 이 성향 덕분에 관계에서 불필요한 감정 소모를 줄이고 편안함을 유지할 수 있습니다.

4-2. 애착 유형별 설명 (강도 기반)

애착 3축 중 가장 높은 점수의 축이 지배적 애착 유형이 됩니다. 점수가 3 이상이면 High, 미만이면 Mid입니다.

애착 유형강도타이틀설명
애착안정High(≥3)💎 견고한 신뢰형타인에 대한 신뢰와 자신에 대한 긍정이 조화를 이룹니다. 갈등을 회피하지 않고 대화로 해결하며, 정서적으로 가장 성숙하고 안정된 관계를 맺습니다.
Mid(<3)🛡️ 안정적 지향형관계에서 오는 안정감을 중요시하며, 파트너를 믿고 지지하려 노력합니다. 비교적 편안하고 지속 가능한 연애를 이끌어갑니다.

Mid(<3)🦋 감성적 교감형사랑받고 있음을 확인받을 때 큰 행복을 느낍니다. 섬세한 감수성을 지녔으며, 파트너와의 정서적 교류를 통해 안정감을 얻습니다.

Mid(<3)🌵 신중한 거리두기형자신의 영역이 존중받길 원합니다. 너무 빠른 친밀감보다는 천천히 신뢰를 쌓아가며, 서로의 삶을 존중하는 균형 잡힌 관계를 지향합니다.
